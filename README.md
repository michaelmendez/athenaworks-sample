# Athenaworks Sample Website

Modern, responsive website built with React, TypeScript, and Ant Design.

## Tech Stack

- React 19 + TypeScript 5.9
- Ant Design 5.28
- Vite + Vitest

## Getting Started

```bash
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm dev      # Start dev server
pnpm build    # Build for production
pnpm test     # Run tests
pnpm lint     # Run linter
```
